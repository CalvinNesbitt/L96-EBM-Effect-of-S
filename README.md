# L96-EBM-Effect-of-S
In this repository we explore the effect of the solar parameter S in a minimal model of climate, the L96-EBM. For more information on the L96-EBM see Gelbrecht et al. 2021 (https://doi.org/10.1140/epjs/s11734-021-00175-0).

The project is split in two main folders: 

- Generating-Attractor-Data. Here we simulate the L96-EBM for different values of S.
- Plotting-Effect-of-S. Here we plot the how the solar parameter S effects the attractor landscape.
